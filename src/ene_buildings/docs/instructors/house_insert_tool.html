Add House Tool

  • Add parametric houses.

Tool Operation

  1. Select template in dialog.
  2. Click viewport to add points defining the house.
  3. Drag cubic handles to set gable angles.
  4. Double click or press enter to save.
  5. Esc = Cancel.

Context Menu

  • Reverse Path - Change on what side of path house is drawn.
  • Add to Path - Toggle whether new corners can be added.
  • Templates - Toggle template dialog visibility.

House Positioning

  • The path should follow plot edge meaning a house facing the street has its facade on it while a house facing the yard usually has a firewall on it.
